# koishi-plugin-bilibili-notify-image

[![npm](https://img.shields.io/npm/v/koishi-plugin-bilibili-notify-image?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-bilibili-notify-image)

`koishi-plugin-bilibili-notify` 的图片渲染插件，基于 Puppeteer 生成推送卡片图片。

> [!NOTE]
> 需要先安装 `puppeteer` 插件
>
> 安装后，直播插件和动态插件会自动使用图片推送

## 功能

- 直播开播 / 直播中 / 下播卡片
- 动态卡片（图文、视频、专栏、转发等）
- SC（超级留言）卡片
- 上舰（大航海）卡片
- 弹幕词云图片
- 自定义卡片渐变背景色、底板颜色、字体

## 安装

在 Koishi 插件市场中搜索 `bilibili-notify-image` 并安装。
